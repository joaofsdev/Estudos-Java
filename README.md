# Estudos Java - DevDojo

Este repositório contém meus estudos de Java do curso **"Maratona Java Virado no Jiraya"** do DevDojo, disponível gratuitamente no YouTube.

## Sobre o Curso

O curso "Maratona Java Virado no Jiraya" é uma formação completa e gratuita de Java, ministrada pelo DevDojo no YouTube. Este repositório serve como um registro do meu progresso e aprendizado ao longo do curso.

## Estrutura do Repositório

- **maratona-java/**: Contém os exercícios e exemplos das aulas do curso

- **projetos-java/**: Projetos práticos desenvolvidos durante os estudos
